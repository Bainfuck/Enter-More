# Enter-More
一个Brainfuck-like的极简主义语言。
基于一条可以左右无限移动的纸带和一个尺寸无限大的栈。


语法表如下：
|指令  |语义  |
|- |- |
|>|左移纸带一格|
|<|右移纸带一格|
|+|纸带当前位置+1|
|-|纸带当前位置-1|
|?|将纸带当前位置值入栈|
|!|弹出栈顶并覆盖到纸带当前位置|
|>>|抛弃栈顶|
|<<[num:int]|将num入栈（如果没有则入栈0）|
|?~|交换栈顶|
|!~|交换纸带当前位置左右一格|
|?~!|将栈顶与纸带当前位置与非运算，结果入栈|
|!~?|将栈顶与纸带当前位置与非运算，结果覆盖到纸带当前位置|
|?!|将纸带当前位置数值输出|
|!?|弹出栈顶数字输出UTF-8表中对应字符|
|X/x|输入字符串，随后对字符串从左向右将每个字符对应的UTF-8数值入栈|
|(|如果纸带当前位置数值为0,跳转到指令流右侧最近的“)”，如不存在就跳转到指令流尾|
|)|如果纸带当前位置数值不为0,跳转到指令流右侧最近的“(”，如不存在就跳转到指令流头|
|[nums:int](|与指令流右侧最接近的“)”共同组成固定周期循环，如右侧“)”不存在则与指令尾构成循环|
|连续\n|指令循环向右移动，移动距离取决于\n的连续长度|
|连续[SPACE]|指令循环向左移动，移动距离取决于\n的连续长度|
|\t|命令分隔符|
